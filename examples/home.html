Welcome to RenX

A lightweight, student‑built browser engine powered by Python + Pygame.

Quick Actions

Open HTML file

Press Ctrl + O or click a link below.

New Tab

Click the + button or press Ctrl + N.

Refresh

Hit the ~ button or Ctrl + R.

Navigate

←Back/→Forward or click the arrow buttons.

Close Tab

Click the x on a tab.

What RenX Understands

  • HTML: div, p, headings h1‑h6, span, br, hr>, <a>, <img> …
  • CSS: color, font‑size/weight/family, margin, padding, border, background, display:block/inline/none, text‑align, images, shorthand expansion.
  • External stylesheets via link rel="stylesheet"
  • Images (PNG/JPG) with automatic size detection if height not provided.
  • Scrolling, multi‑tab UI, link navigation, back/forward history, refresh.
  • Dynamic cursor change on link hover.

⚡ More tags and properties on the way-stay tuned!
